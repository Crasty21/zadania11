# Lista zakupów – jQuery

## Opis projektu
Aplikacja „Lista zakupów” została wykonana z użyciem HTML, CSS oraz biblioteki jQuery.
Umożliwia dynamiczne zarządzanie listą produktów wraz z efektami wizualnymi
i zaawansowanymi operacjami na DOM.

## Funkcje aplikacji
- dodawanie produktu do listy
- dodawanie produktu na początek i na koniec listy
- usuwanie ostatniego produktu
- czyszczenie całej listy
- przywracanie przykładowej listy
- edycja elementu listy po kliknięciu (input + Enter)
- efekty fadeIn i fadeOut podczas edycji
- podświetlanie klikniętego elementu
- kolorowanie co drugiego elementu listy
- sortowanie alfabetyczne A–Z
- filtrowanie listy po wpisanej frazie
- przeciąganie i zmiana kolejności elementów (drag & drop)

## Użyte technologie
